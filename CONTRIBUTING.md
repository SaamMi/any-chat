🛠 Pull Request Process
Create a Branch: Use a descriptive name like feat/file-uploads or fix/widget-alignment.

Write Tests: If you are adding a feature, please add a corresponding test in tests/Unit or tests/Browser.

Update Documentation: If your change changes how a prop works (e.g., primaryColor), update the README.md.

Submit the PR: Provide a clear description of the changes and link any related issues.
